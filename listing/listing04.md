Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа поочереди выведит цифры, но когда цикл в горутине закончится то случится дедлок, потому что цикл в майне не знает когда остановиться. Что бы этого избежать нужно закрыть канал перед завершением горутины

```
