Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
вывод: [77 78 79]
переменная b представляет собой срез по массиву а
по правилам среза вторая цифра не входит в сам срез

```
